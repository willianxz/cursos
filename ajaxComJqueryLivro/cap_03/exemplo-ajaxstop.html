Este é o conteúdo requisitado para demonstrar o método ajaxStop( )

O evento associado ocorre quando termina uma requisiçao AJAX.

Este método (em terminologia jQuery vamos chamar de métodos) admite um argumento facultativo que é uma função callback a ser executada por ocasião da ocorrência do evento.

Exemplo:

  1.  $('h1').ajaxStop(function() {
  2.    $(this)
  3.    .html('Conteúdo inserido com uso do método ajaxStop()')
  4.    .css('color', '#c30');
  5.  });
