Este é o conteúdo requisitado para demonstrar o método ajaxStart( )

O evento associado ocorre quando se inicia uma requisiçao AJAX e nenhuma outra atividade está em curso na página, como por exemplo: outra requisição em andamento.

Este método (em terminologia jQuery vamos chamar de métodos) admite um argumento facultativo que é uma função callback a ser executada por ocasião da ocorrência do evento.

Exemplo:

  1.  $('#carregando').ajaxStart(function() {
  2.    $(this).show('slow');
  3.  });
